# AGENTS.md

This repo is a premium beginner stock-learning product.

Rules:
- Educational only, never financial advice.
- Not a stock screener, dashboard, or trading signal app.
- One primary action per screen.
- Keep UI minimalist, premium, warm, and uncluttered.
- Follow the Figma design as the visual source of truth.
- Do not invent a new design system.
- Keep course flow strict: Learn -> Practice -> Check -> Reward.
- Wrong answers should feed a lightweight review queue.
- Leaderboard updates only at milestones.
- Nickname should be stored locally and used naturally.
- Certificate printing must use a dedicated print-safe view or print stylesheet.
- Never add duplicate navigation paths or redundant CTAs.
- you are able to use Figma Implement Design and Figma skill
